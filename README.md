# Transportation-Cost-Analysis-Across-Nigeria-2024-
This project presents an interactive Microsoft Excel Dashboard analyzing transportation fares across the 36 states of Nigeria using data published by the National Bureau of Statistics (NBS)

The analysis focuses on transportation costs for:

✈️ Air travel (specified routes – single journey)
🚌 Bus intercity transportation

The dashboard was designed to provide stakeholders with a quick overview of transportation cost trends, enabling comparisons across states and months.

Project Objectives

The objectives of this project were to:

Analyze transportation fare patterns across Nigeria.
Compare monthly air and bus transportation costs.
Build an interactive dashboard for easy data exploration.
Demonstrate data cleaning, analysis, and visualization skills using Microsoft Excel.

Data Source

National Bureau of Statistics (NBS), Nigeria

Dataset:
Transportation Fare Watch (2024)

The dataset contains monthly transportation fare information collected from the 36 states and the Federal Capital Territory (FCT).


Tools Used

Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Data Analysis
